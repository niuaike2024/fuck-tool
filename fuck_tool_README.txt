命令行智能修正工具 (Fuck Command Tool)

=== 功能介绍 ===
当你在命令行中输入错误的命令后，只需输入'fuck'，这个工具将自动尝试修正并执行正确的命令。

主要功能：
1. 自动识别上一条输错的命令
2. 修正常见的命令拼写错误（如ls → Get-ChildItem, cd → Set-Location等）
3. 当检测到权限错误时，提示以管理员身份运行
4. 找不到命令时，会列出相似的命令供选择
5. 在执行修正后的命令前会进行确认

=== 安装说明 ===
方法一：临时使用（无需安装）
1. 确保fuck.ps1和fuck.bat文件都在桌面
2. 在命令提示符(cmd)中，直接运行：fuck

方法二：永久安装（推荐）
1. 将fuck.ps1和fuck.bat文件复制到一个固定目录，如C:\Tools\
2. 将该目录添加到系统环境变量Path中：
   - 右键点击"此电脑" → 属性 → 高级系统设置 → 环境变量
   - 在系统变量中找到Path，点击编辑
   - 点击新建，添加C:\Tools\（或你的实际目录）
   - 点击确定保存
3. 重启命令提示符，现在可以在任何目录下使用fuck命令了

=== 使用方法 ===
1. 尝试输入一个错误的命令，例如：ls（在cmd中这不是内部命令）
2. 然后输入：fuck
3. 工具会分析错误并提供修正建议
4. 确认后会执行修正后的命令

=== 注意事项 ===
- 首次使用PowerShell脚本可能需要设置执行策略：
  在PowerShell中运行：Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
- 该工具仅在Windows系统的命令提示符(cmd)和PowerShell中有效
- 使用管理员权限运行命令时，会弹出用户账户控制(UAC)提示

=== 扩展建议 ===
你可以修改fuck.ps1文件中的修正规则，添加更多自定义的命令修正模式。
